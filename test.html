Installation Instructions

  1. Download all extension files to a folder on your computer
  2. Open Chrome and go to chrome://extensions/
  3. Enable "Developer mode" (toggle in top right)
  4. Click "Load unpacked" and select your folder
  5. Go to LinkedIn.com and look for the extension icon in your toolbar

2. Extension Files

Make sure you have downloaded all these files:

  • manifest.json - Extension configuration
  • popup.html - Extension popup interface
  • popup.css - Popup styling
  • popup.js - Popup functionality
  • background.js - API communication
  • content.js - LinkedIn page interaction
  • config.js - Configuration settings

3. Usage Tips

  • Navigate to LinkedIn first: Make sure you're on linkedin.com/feed or your LinkedIn homepage
  • Start small: Begin with 1-2 posts to test the functionality
  • Monitor activity: Watch the activity log in the extension popup
  • Realistic limits: Don't exceed 10-15 comments per session to avoid rate limiting
  • Quality over quantity: The AI generates thoughtful comments, but always review them

4. Troubleshooting

  • Extension not showing: Make sure Developer mode is enabled
  • API errors: Check your OpenAI API key and account credits
  • Posts not found: Try refreshing LinkedIn and scrolling down to load posts
  • Comments not posting: LinkedIn may have updated their interface - check the activity log
